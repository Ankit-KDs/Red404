# 🩸 Red404 - RealTime Blood Donation Platform

## What is Red404?

Red404 is a **smart blood donation platform** that connects blood donors with hospitals in real-time. When a hospital urgently needs blood, Red404 instantly finds the nearest available donors and sends them a request.

**Problem it solves:** Blood shortage in emergencies  
**Solution:** Connect donors with hospitals using location-based matching

---

## Who Uses Red404?

### 👨‍💼 Donors (Blood Givers)
- Register your blood type and location
- Get notifications when hospitals need your blood
- Accept or decline blood donation requests
- Track your donation history

### 🏥 Hospitals (Blood Seekers)
- Find donors near you within minutes
- Send urgent blood requests
- See donor details and distance
- Track request status in real-time

## 💻 How It Works

### For Hospitals:
```
1. Hospital logs in
2. Searches for donors by blood type
3. Sets search radius (10 km, 20 km, etc.)
4. Sees list of nearby available donors
5. Sends urgent blood request
6. Waits for donor response
7. Confirms donation when accepted
```

### For Donors:
```
1. Register with blood type and location
2. Set yourself as "Available"
3. Receive request notification
4. Read hospital details
5. Accept or decline request
6. Hospital confirms donation details
7. History gets recorded
```

## 🚦 Getting Started

### Prerequisites
- Java 11 or higher
- MySQL database
- Web browser (Chrome, Firefox, Safari, Edge)

### Installation
1. Clone this repository
2. Setup MySQL database
3. Configure database connection
4. Build with Maven: `mvn clean package`
5. Deploy on Tomcat server
6. Visit: `http://localhost:8080`

## ❓ FAQ

**Q: Is my blood type private?**
